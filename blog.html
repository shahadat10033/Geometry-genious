Some Important Question & Answers

*Q1 - What is DOM?And What is the purpose of DOM??


*A1- The word DOM means Document Object Model. It is usually used to add, remove, and modify elements of a website. Basically it gives interactivity to website .


*Q2 - How will you select HTML elements using DOM?Name the DOM methods.

*A2- we will select HTML elements by some DOM methods. Name of the DOM method :

1. getElementById()
2.getElementsByClassName()
3. getElementByTagName()
4. querySelector()
5. querySelectorAll()


*Q3 - What is event bubble?

*A3- A bubble event occurs when we click at the lower div and the event “bubbles” to its immediate upper element and then climbing higher from higher is called event bubbles


*Q4 - What is callback function and why will you use it?

*A4- A call back function is a function that is passed as a parameter to another function.when we request data from other sources,we use this function.
